## üéØ Common Questions

**Q: Do I need an API key?**  
A: No! It works without LLM - just generates prompts you can use manually.

**Q: Where is my data stored?**  
A: Locally in the `data/` folder. 100% private.

**Q: Can I customize it?**  
A: Absolutely! Everything is customizable.

**Q: How do I add my own decisions?**  
A: `twin.add_decision("decision", "reason", tags=["tag"])`

**Q: What if I want AI responses?**  
A: Add your API key to `.env` and use `examples/llm_integration.py`
